# QuantHedger

**RL-based Derivatives Hedging Engine** — Monte Carlo backtesting, Greeks Explorer, Stress Testing, Strategy Comparison, and Live P&L Tracker.

QuantHedger is a web application for exploring, comparing, and stress-testing derivatives hedging strategies. It is designed to help users simulate portfolio behavior under different market conditions and evaluate hedging performance through interactive analytics.

## Features

- Monte Carlo backtesting for strategy evaluation.
- Greeks Explorer for examining option sensitivity.
- Stress testing for adverse market scenarios.
- Strategy comparison across multiple hedging approaches.
- Live P&L tracking and portfolio monitoring.
- Clean, responsive UI built with modern frontend tooling.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
git clone https://github.com/grisheet/QuantHedger.git
cd QuantHedger
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Open the app locally or through the live demo.
2. Navigate between hedging tools and analytics panels.
3. Compare strategies and inspect portfolio behavior.
4. Use stress tests and backtests to evaluate risk.
